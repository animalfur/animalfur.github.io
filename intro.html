Introduction

Felting is a truly unique craft. People started felting as long ago as 6300 BCE. Wool, which you can use to felt, can absorb water,
which makes it very useful. Also, you only need a needle, wool, and a felting pad to needle felt. This makes it a very cheap
hobby. Another useful quality about wool is that it can be cut cleanly, without fraying it.
You can buy clean wool from the store to felt, or or you can just clean it yourself. Felting is an fascinating project,
and it includes instructions on needle felting, animal fur in the past, and how it works!
